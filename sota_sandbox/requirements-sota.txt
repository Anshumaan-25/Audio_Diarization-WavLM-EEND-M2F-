# Target box: Ubuntu 22.04, NVIDIA RTX 6000 Ada (CUDA 12.x), Python 3.10.
#
# Install (on the Ubuntu machine):
#   python3.10 -m venv .venv-sota
#   source .venv-sota/bin/activate
#   pip install --upgrade pip
#   # 1) CUDA build of torch/torchaudio from the cu121 wheel index:
#   pip install torch==2.1.2 torchaudio==2.1.2 \
#       --index-url https://download.pytorch.org/whl/cu121
#   # 2) the rest from PyPI:
#   pip install -r requirements-sota.txt
#
# torch/torchaudio are listed below (pinned) for the record, but MUST be
# installed from the cu121 index above to get CUDA wheels — plain PyPI gives
# CPU-only builds. The version pins match the pipeline's torch (2.1.2+cu121)
# so the two stacks could even share a wheel cache, though the venvs stay
# separate by design.

# --- torch stack (install from the cu121 index — see header) -----------------
